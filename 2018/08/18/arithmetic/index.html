1，判断某序列是否是二叉搜索树的后续遍历序列

  • 首先要看二叉搜索树的特性，及左边比根节点小，右边比根节点大
  • 先取序列的最后一个元素，即是该二叉搜索树的根节点，然后从头遍历该序列，知道某一元素比根节点大，然后从该节点开始，遍历如果出现比根节点小的数，那么就可以判断该序列不是二叉搜索树的后序遍历，依次遍历根节点的左子树和右子树，得出结果！

2，求二叉树中所有路径值的和为某一整数的所有路径

  • 首先一条路径都是从根路径开始的，所以我们可以从根节点开始遍历
  • 从根节点开始遍历的，我们可以想到前序遍历，这里我们采用前序遍历，先从根节root点开始，然后遍历root.left 然后节点值相加如果满足条件则该路径就是一条要查找的路径，一直遍历到叶子节点，如果都没有满足条件则递归返回父节点，然后遍历父节点的右子节点，再返回父节点的时候要删除序列中父节点的子节点值，然后依次递归！

3,复制复杂链表，该链表有指向下一个节点的next指针和指向任意节点的sibling指针

  • 分为三步，首先复制原始链表，把复制链表连接到原始链表后面，比如A->B->C->D 复制后变为A->A1->B->B1->C->C1->D->D1
  • 然后A1.sibling = A.sibling.next这样我们就可以复制完随机链表，现在复制链表和随机链表在一个链表上
  • 第三步，然后把链表拆分，奇数上的是原始链表，偶数上的是复制链表。

4，数组中出现次数超过一半的数字

  • 数组中一个数字出现的次数超过数组长度的一半，说明这个数字出现的次数比其它数字出现次数的和还要大。我们在遍历数组的时候可以保存两个值，第一个是数字的值，第二个是数字出现的次数。保存第一个数字的值，给定出现次数为1，如果下一个数字和之前保存的数字相同，则次数加1，否则次数减1。如果次数为0，就保存下一个数字，且给次数赋值为1。因为要找的数字出现次数比其它数字出现次数的和还要大，所以最后保存的值以定就是我们要找的值。。。。。
  • 比如：数组{1，2，3，2，2，2，5，4，2} 要找的数字就是2！
    1. 我们在遍历的时候，先给定result = 1，times = 1
    2. 遍历到2的时候，result = 2，times = 0 所以这时候就要给result和times赋值,result为下个值及 result = 3，times = 1
    3. 依次遍历，最后的result就是我们要求的数字！

6，数组中只出现一次的两个数字

#####一个整型数组中除了两个数之外，其它数都出现了二次，找出这两个只出现一次的数字###

  • 除了要找的两个数字，其它数都出现两次，可以使用异或^
  • A异或A等于0，A异或0等于A
  • 由此我们可以，依次异或数组的所有数，出现两次的数异或就为0
  • 得到一个二进制数，该二进制中为1的位数肯定大于等于1，找到其中一个为1的坐标index
  • 把数组分为index为0和index为1的两个数组，再分别异或及得到要找的两个数字

例如：输入数组[2,4,3,6,3,2,5,5],依次异后得到二进制0010,该二进制中倒数第2个数字为1,于是根据二进制倒数第2个位置是否为1，把数组分为[2,3,6,3,2]二进制倒数第2都是1和[4,5,5]倒数第二个都是0，然后两个数组分别异或即位要求的只出现一次的数。

11,扑克牌中的顺子

从扑克牌中随机抽取5张牌，判断是不是一个顺子，即这5张牌是不是连续的，2-10为数字本身，A为1，J为11，Q为12，K为13，大小王可以看成任意数字。

  • 我们把大小王看成数字0,这样不会与其他数字冲突，
  • 把抽取的5张牌放入数组中，把数组排序
  • 遍历查找数组中0的个数numberOfZero，即大小王的个数，
  • 遍历查找相邻数字之间的差值numberOfGap
  • 因为大小王可以表示任意数字，只要numberOfZero>=numberOfGap，就代表大小王可以填补所有不连续数字
  • 实时上numberOfZero >= 2

14，求字符串的最长不重复子串

例：输入abcabcbb，输出3，因为abcabcbb最长不重复字串为abc，长度为3.

  • 最长不重复子串，定义一个HashSet数据结构存储数据
  • 从头遍历字符串，如果HashSet里没有，就添加到HashSet中，记录每次HashSet与max的最大值。
  • 如果HashSet中有这个字符，从头开始遍历字符串，从HashSet中删除遍历的字符直到HashSet中没有这个字符。
